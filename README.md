\## Overview



Traditional vehicle insurance premiums rely primarily on static attributes such as driver history and vehicle age. This research proposes a decentralized framework that dynamically adjusts insurance premiums using real-time vehicle health information collected through On-Board Diagnostic (OBD) systems.



The proposed framework integrates:



\* Machine Learning for predictive maintenance

\* Ethereum Smart Contracts

\* Chainlink Decentralized Oracle Network

\* Flask API for off-chain prediction serving

\* Web3-based insurance interface



The system enables transparent, tamper-resistant, and data-driven insurance premium calculation based on actual vehicle condition.
